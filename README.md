# 📚 Books Web Scraper (Python Project)

## 📌 Project Overview
This project extracts book data from an online website using Python and BeautifulSoup.

## 🎯 Objectives
- Scrape book details from multiple pages
- Extract title, price, and rating
- Store data in CSV format

## 🛠 Tools Used
- Python
- BeautifulSoup
- Requests
- CSV

## 📊 Features
- Multi-page scraping
- Error handling
- Clean data storage

## 📁 Output
- CSV file containing:
  - Book Title
  - Price
  - Rating

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/webscrapping2.py
```

## 💡 Use Case
This project demonstrates how web scraping can be used for:
- Price comparison
- Market research
- Data collection

## 🚀 Conclusion
The script successfully collects structured data from multiple pages and stores it for further analysis.
